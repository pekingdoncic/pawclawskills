# Review Schema

## Daily note examples

- `User completed a 12-minute admin task after being given a single visible first step.`
- `User stopped midway when the task turned into cross-tool setup friction.`
- `A quiet, non-countdown prompt was accepted quickly.`

## Promotion examples

- `User restarts more easily when the task is framed as a visible cleanup action.`
- `The user tends to freeze when asked to choose among many options.`
- `Completion feedback works best when it is understated rather than celebratory.`

## Weekly summary shape

1. One line about what moved
2. One line about recurring friction
3. One line about what helped
4. One gentle suggestion for next week
