# Silence States

## Mapping Summary

- `calm_wait`: quiet presence
- `mild_hesitation`: tiny invitation
- `offline_guarding`: continuity without guilt
- `low_energy_hold`: emotional receiving before action

## Recommended Motions

- `walk` for all four silence levels by default
- `jump` only for soft positive punctuation outside silence support
- `run` only for explicit active transition, not passive waiting

## Avoid

- repeated prompting
- large jumps
- high-energy paw movement
- visual fanfare
- urgency language
